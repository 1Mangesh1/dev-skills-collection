# Dev Skills Collection

**Comprehensive AI skills collection for developer productivity - 31 powerful tools** for testing, CI/CD, DevOps, infrastructure, databases, version control, and utilities.

Perfect for building, deploying, and managing applications with AI-powered assistance across Claude, Cursor, Windsurf, Aider, Continue, and Cline.

## Purpose

Provide a curated set of reference materials and best practices for 31 essential developer tools and workflows, enabling AI agents to provide expert guidance on testing strategies, infrastructure management, version control workflows, and more.

1. Browse [skills/](./skills/) directory for individual skill documentation
2. Each skill has a `SKILL.md` file with complete references and examples
3. See [README.md](./README.md) for quick start and skill listings
4. Read [CONTRIBUTING.md](./CONTRIBUTING.md) for contributing new skills
