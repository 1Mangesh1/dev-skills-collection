# AI Prompting Quick Start

Master prompt engineering techniques for Claude, GPT, and other LLMs.

## Quick Reference

### Basic Prompt Template
```
Role: [Define your expertise]
Task: [Clear, specific request]
Context: [Relevant background]
Format: [Expected output format]
Constraints: [Any limitations]
```

### Key Techniques
- **Few-shot examples** - Show 2-3 examples first
- **Chain-of-thought** - Ask to explain reasoning step-by-step
- **Role-playing** - Specify a persona/expertise level
- **Structured output** - Request specific formats (JSON, markdown, etc.)
- **Constraints** - Define length, tone, complexity

## Common Patterns

1. **Classification** - Ask model to categorize inputs
2. **Generation** - Creative writing, code, content
3. **Analysis** - Break down problems, explain concepts
4. **Translation** - Convert between languages or formats
5. **Extraction** - Pull specific info from text
